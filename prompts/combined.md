# Combined Logo (Combination Mark)

## Description
A combined logo integrates a symbol/icon with the brand name in a unified composition. It is the most versatile type of logo. Examples: Adidas, Burger King, Lacoste, Doritos, Mastercard.

## Prompt Guidelines

### Structure
- Symbol/icon + brand name working together
- The symbol and text must work as a unit
- They can be separated for independent use
- Clear visual hierarchy between icon and text

### Possible Arrangements
- **Horizontal**: Icon on the left, text on the right (most common)
- **Vertical (stacked)**: Icon above, text below
- **Integrated**: Icon incorporated into the text
- **Enclosed**: Both within a container

### Symbol Design
- Must complement the typography, not compete with it
- Proportional scale to the text
- Can represent the business literally or abstractly
- Simple and recognizable shapes

### Typography
- Must harmonize with the icon style
- If the icon is geometric, the font should follow that language
- Typographic weight balanced with the icon's visual weight
- Kerning adjusted for visual integration

### Composition
- Precise alignment between icon and text
- Consistent and proportional spacing
- Balanced visual center of gravity
- Construction grid with defined proportions

### Best Practices
- Work with and without the symbol
- Legible at small sizes
- Work in a single color
- Maintain proportions across different applications
- Compact version and expanded version

### AI Instructions
Create a professional combined logo that harmoniously integrates a symbol/icon with the brand name. The symbol and typography must share the same visual language. Clean and versatile design on a transparent background.
