# Regra de Design: Grid Circular

## Princípio
O grid circular usa círculos e arcos como base para construção do logo, garantindo harmonia orgânica e equilíbrio visual. É uma das técnicas mais usadas em design de logos profissionais.

## Aplicação no Logo

### Construção
- Use círculos de diferentes tamanhos como guias para curvas e contornos
- Arcos de círculos definem bordas e transições suaves
- Interseções de círculos criam pontos de ancoragem naturais
- Tangências entre círculos definem alinhamentos orgânicos

### Proporções
- Círculos com proporções baseadas em fibonacci (1, 1, 2, 3, 5, 8...)
- Cada elemento deve se alinhar a pelo menos um círculo do grid
- Raios dos círculos guiam espessuras de traços
- Espaçamentos derivados dos diâmetros dos círculos

### Benefícios
- Curvas suaves e naturais
- Harmonia visual intrínseca
- Consistência entre elementos
- Base matemática para justificar decisões de design

### Instrução
O logo deve ser construído usando um grid circular como base. Todas as curvas devem seguir arcos de círculos. As proporções devem derivar de relações geométricas circulares. O resultado deve ter fluidez orgânica e precisão matemática.
