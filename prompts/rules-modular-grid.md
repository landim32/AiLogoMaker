# Design Rule: Modular Grid

## Principle
The modular grid divides space into equal modules (basic units) that serve as the basis for positioning and sizing all logo elements. It ensures consistency, alignment, and scalability.

## Application to Logo

### Base Unit
- Define a minimum unit (module) as the foundation for the entire design
- All elements are multiples of this unit
- Stroke thickness = 1x or 2x the unit
- Spacing = multiples of the unit

### Grid
- Divide the space into a regular grid (e.g., 24x24, 32x32, 48x48)
- All vertices and anchor points snap to the grid
- Curves are tangent to grid lines
- No element falls "between" grid points

### Modularity
- Elements can be composed of repeated modules
- Patterns emerge from unit repetition
- Scaling maintains grid integrity
- Adaptation to different sizes preserves proportions

### Protection Area
- Define a safety margin around the logo
- Margin = multiple of the base unit (usually 2x-4x)
- No external element invades this area
- Consistency across all applications

### Instruction
The logo must be built on a strict modular grid. All elements must align perfectly to the grid. Use a consistent base unit for all measurements. The result must be precise, ordered, and perfectly scalable.
