# Emblem

## Description
An emblem is a logo where the text is contained within a symbol, seal, or crest. It conveys tradition, authority, and prestige. Examples: Starbucks, Harley-Davidson, BMW, NFL, universities.

## Prompt Guidelines

### Structure
- Text integrated within a decorative shape/container
- Generally circular, oval, or shield format
- Decorative elements such as borders, ribbons, crests
- Symmetric and centered composition

### Types of Emblem
- **Seal/Badge**: Circular with surrounding text (e.g., Starbucks)
- **Shield/Crest**: Heraldic shield format (e.g., Porsche)
- **Crest**: Combined heraldic elements (e.g., universities)
- **Modern badge**: Geometric shapes with integrated text

### Design
- Container shape with defined borders
- Text following the container's curvature
- Central focal element (icon, illustration, symbol)
- Layers of information (top text, symbol, bottom text)
- Decorative details that enrich without overwhelming

### Typography
- Fonts that evoke tradition and quality
- Text can follow curved paths
- Hierarchy between main name and secondary elements
- Classic serif or geometric sans-serif

### Composition
- Radial or bilateral symmetry
- Multiple concentric layers
- Balance between positive and negative space
- Golden ratio in internal elements

### Best Practices
- Work on stamps, embroidery, and engraving
- Maintain legibility when reduced
- Simplified version for smaller sizes
- Work in a single color
- Convey authority and tradition
