# Regra de Design: Grid Modular

## Princípio
O grid modular divide o espaço em módulos iguais (unidades básicas) que servem como base para posicionar e dimensionar todos os elementos do logo. Garante consistência, alinhamento e escalabilidade.

## Aplicação no Logo

### Unidade Base
- Definir uma unidade mínima (módulo) como base de todo o design
- Todos os elementos são múltiplos dessa unidade
- Espessura de traços = 1x ou 2x a unidade
- Espaçamentos = múltiplos da unidade

### Grid
- Dividir o espaço em uma grade regular (ex: 24x24, 32x32, 48x48)
- Todos os vértices e pontos de ancoragem se encaixam no grid
- Curvas tangenciam as linhas do grid
- Nenhum elemento fica "entre" pontos do grid

### Modularidade
- Elementos podem ser compostos por módulos repetidos
- Padrões emergem da repetição de unidades
- Escalamento mantém a integridade do grid
- Adaptação a diferentes tamanhos preserva proporções

### Área de Proteção
- Definir margem de segurança ao redor do logo
- Margem = múltiplo da unidade base (geralmente 2x-4x)
- Nenhum elemento externo invade essa área
- Consistência em todas as aplicações

### Instrução
O logo deve ser construído sobre um grid modular rigoroso. Todos os elementos devem se alinhar perfeitamente ao grid. Use uma unidade base consistente para todas as medidas. O resultado deve ser preciso, ordenado e perfeitamente escalável.
