# Design Rule: Circular Grid

## Principle
The circular grid uses circles and arcs as the basis for logo construction, ensuring organic harmony and visual balance. It is one of the most used techniques in professional logo design.

## Application to Logo

### Construction
- Use circles of different sizes as guides for curves and contours
- Circle arcs define edges and smooth transitions
- Circle intersections create natural anchor points
- Tangencies between circles define organic alignments

### Proportions
- Circles with proportions based on Fibonacci (1, 1, 2, 3, 5, 8...)
- Each element must align to at least one circle in the grid
- Circle radii guide stroke thicknesses
- Spacing derived from circle diameters

### Benefits
- Smooth and natural curves
- Intrinsic visual harmony
- Consistency between elements
- Mathematical basis to justify design decisions

### Instruction
The logo must be constructed using a circular grid as the base. All curves must follow circle arcs. Proportions must derive from circular geometric relationships. The result must have organic fluidity and mathematical precision.
