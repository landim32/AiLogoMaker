# Mascot (Mascot Logo)

## Description
A mascot logo uses an illustrated character as the brand's visual representative. It creates emotional connection and personality. Examples: KFC (Colonel Sanders), Michelin (Bibendum), Pringles, Duolingo (Duo).

## Prompt Guidelines

### Structure
- Illustrated character as the central element
- May include the brand name alongside the character
- The character must have personality and expression
- Stylized design, not realistic

### Character Design
- Rounded and friendly shapes
- Clear and charismatic facial expression
- Dynamic or welcoming posture
- Stylized proportions (large head, smaller body)
- Limited and harmonious color palette
- Enough detail for personality, without excess

### Types of Mascot
- **Stylized human**: Cartoon human character (e.g., KFC)
- **Anthropomorphized animal**: Animal with human characteristics (e.g., Duolingo)
- **Creature/Object**: Fictional object or creature (e.g., Android, Michelin)
- **Abstract character**: Shape with personality (e.g., Snapchat ghost)

### Composition
- The character must be the visual focus
- Proportion that allows use as an icon
- Unique and recognizable silhouette
- Harmonious integration with typography when present

### Best Practices
- Work at various sizes
- Maintain personality even when simplified
- Have full body and face/bust versions
- Work in a single color
- Be versatile for animation and different poses
- Create empathy and emotional connection
